Instructions to use:

python server.py ServerIP PortToBindTo
python client.py ServerIP ServerPort

You will notice a log has been created. This is logging some information which can be analyzed using the analyze.py (which can be found in the report folder)

python analyze.py pathToLogFile
This will generate a file report_info.txt with the analysis.

P.S. Histogram image provided separately as well as report image might not be clear due to resizing.
